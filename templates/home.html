About the System

The NetFlask Recommender Engine uses a collaborative filtering algorithm to recommend movies. The engine takes ratings already existing in the database (provided by other users) and ratings provided by the user him/herself to rank movies according to a weighted Pearson Correlation Score and return a list of movies that the engine predicts the user will enjoy.
